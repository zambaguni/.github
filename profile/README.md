## 👋 About Us

ALA 팀은 더 나은 서비스를 만들기 위해 함께 성장하는 개발 팀입니다.

우리는 **백엔드(Java)**와 **프론트엔드(TypeScript)**를 기반으로 서비스를 개발하고 있으며,
깔끔한 코드와 효율적인 협업을 지향합니다.
